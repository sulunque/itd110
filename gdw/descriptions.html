HTML Terms

HTML
This markup tells a web browser how to display text, images and other forms of multimedia on a webpage.
HTML Element
The HTML element is everything from the start tag to the end tag.
HTML Tag
The html tag tells the browser that this is an HTML document.
Angle Bracket
Visual Basic uses angle brackets to define attributes that add metadata to code.
HTML Attribute
HTML attributes are special words used inside the opening tag to control the element's behaviour.
HTML Comment
used to insert comments in the source code.
HTML Object Element
tag defines an embedded object within an HTML document.
HTML Special Character
special characters is a selection of punctuation characters that are present on standard US keyboard and frequently used in passwords.
